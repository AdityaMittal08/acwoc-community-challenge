# AcWoC Community Connect Challenge

Welcome to the **AcWoC Community Connect Challenge** repository.

This repository hosts a **special one-time community challenge** for **AcWoC 2026** participants. Completing this challenge successfully will earn you **+150 bonus points** on the official AcWoC leaderboard.

## üìå Steps to Earn 150 Points

### Step 1: Follow AcWoC on Social Media

Follow **both** of the official AcWoC social media pages:

* Instagram
* LinkedIn

---

### Step 2: Take Screenshots

Take clear screenshots showing that you are following:

* [Android Club Instagram page](https://www.instagram.com/androidclub.vitb/)
* [Android Club LinkedIn page](https://www.linkedin.com/company/android-club-vit-bhopal/)

Make sure your **username and follow status are clearly visible** in the screenshots.

---

### Step 3: Create a Pull Request

1. Fork this repository
2. Create a new folder with **your GitHub username** as the folder name
3. Add your screenshots inside that folder
4. Commit and push your changes
5. Open a Pull Request to this repository

---

### Step 4: Maintainer Verification

* Maintainers will review your Pull Request
* If everything is valid, your PR will be **merged**
* A special label **`community-connect`** will be added by maintainers

Once the PR is merged with the label, **150 points will be automatically added** to your leaderboard score.
